# HabitKeeper — 产品决策文档

> 把"差评 → 痛点 → 竞品定位 → 卖点 → MVP"的决策链固化下来。所有数字来自
> [play-pain-miner](https://github.com/94xhn/play-pain-miner) 对 Google Play 习惯打卡品类
> 1–2 星差评的真实挖掘(2026-06,3 个应用 / 120 条差评)。

## 1. 竞品定位

| 角色 | App | 性质 | 机会指数 | 用途 |
|---|---|---|---|---|
| 🎯 主攻假想敌 | HabitNow | 闭源·订阅·500万装机 | 2.02(最高) | 盘子最大、痛点都能解,主攻它的用户池 |
| 🪤 反面靶子 | HabitKit | 闭源·付费墙最狠 | 1.36 | "全功能免费"卖点的对比素材 |
| 📐 功能基线 | Loop (uhabits) | 开源免费·功能扎实 | 1.95 | 要追平的"免费天花板",非商业对手 |

> 机会指数 = log10(安装量) × (5 − 评分)。

**一句话定位:** 像 Loop 一样免费可靠,但比 HabitNow / HabitKit 不坑钱、不坑数据。

## 2. 痛点 → 卖点映射(差评驱动)

| # | 卖点 | 打的痛点(加权热度) | 差评原声证据 |
|---|------|------|------|
| 1 | 灵活周期:每天/每周N次/自定义间隔/可改期 | 功能缺失 `333`(最高) | 79👍 reschedule recurring、11👍 weekly habits、5👍 repeat N times/week |
| 2 | 全功能免费 / 一次买断,不设付费墙 | 收费订阅 `71` | HabitKit 8👍 "stats/charts 要订阅、4个习惯上限"、5👍 "widget 不付费不能用" |
| 3 | 永不丢数据:本地优先 + 自动备份 / 导出到自己的网盘 | 数据丢失 `45` | Loop 3👍 "整年数据全丢 No Cloud Backup"、9👍 "只保留 1 年数据" |
| 4 | 可靠通知:通知带"跳过/否",不逼你假打卡 | 客服无响应 `43` | Loop 21👍 "need a 'no' option"、25👍 "通知突然不弹了" |
| 5 | 简单不混乱:习惯分组 + 拖拽排序 | 难用 `56` | 10👍 "drag and reorder"、3👍 "no way to group habits" |
| — | 零权限、无追踪 | 隐私权限 `106` | 多条抱怨权限/隐私 |

> 选入标准:同类请求在差评中出现 ≥3 次或单条高赞。不是拍脑袋,是 120 个用户用脚投票。

## 3. 商业模式:完全免费 + 开源(非 SaaS、非买断)

差评里被骂最惨的恰恰是 SaaS 那一套,所以我们反向选择:

| SaaS 标配 | 被骂的痛点 | 我们的选择 |
|---|---|---|
| 订阅付费 | 收费订阅 71 | 完全免费,无内购、无广告 |
| 云端托管数据 | 数据丢失 45、同步翻车 | 数据存设备,无服务器 |
| 强制账号登录 | 隐私权限 106 | 零账号可用 |

**为什么免费开源(而非买断):** 作为第一个产品,首要目标是跑通"差评→产品→开源上线"全流程 + 求职作品集,100% 兑现"全功能免费"卖点(对标开源标杆 uhabits)。变现接口(一次性解锁增值)以后想加随时能加,代码结构不冲突。

**架构含义:不需要后端。** 纯客户端 Android app + 用户自己网盘做可选备份(无"服务关停=app变砖"风险)。

## 4. MVP 范围

**v0.1（当前骨架,已落地）**
- domain:Frequency 四类型 + Schedule(is-due 引擎)+ Streak(连续天数,非due日不断)
- data:Room 持久化(habits / habit_logs / habit_groups)+ Repository 映射
- ui:Compose 首页(习惯列表 + 当日打卡 + streak 展示)
- 测试:Schedule / Streak / freq 编解码 JVM 单测

**v0.2（路线图,按卖点优先级）**
- [ ] 卖点4:WorkManager 可靠提醒 + 通知"跳过/否"动作(API 33+ 运行时申请 POST_NOTIFICATIONS)
- [ ] 添加/编辑习惯界面(选频率、emoji、分组)
- [ ] 卖点3:导出/导入 + 自动备份到用户网盘
- [ ] 统计:完成率、日历热力图、最长 streak
- [ ] 卖点5:习惯分组 + 拖拽排序
- [ ] 桌面 widget(Glance):打卡按钮可用、显示 streak(直接打 HabitKit "widget 要付费"的脸)

## 5. 验证回路

做出原型后,回到 HabitNow / Loop / HabitKit 的 Play 差评区——那些抱怨上述痛点的用户就是
种子用户名单,他们已经明确说过想要什么。
